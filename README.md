# A humble Fastify boilerplate

Simple Fastify boilerplate defines a simple structure to get started.

## Getting Started

### Dependencies

* Node 14+

### Installing

Clone the repo then

```bash
npm i
```

### Executing program

```bash
npm start
```

Server listens in port 3000 and provides a `/login` route and a `/info`
